## What I Learned

In this project, I practiced using Flexbox for horizontal and vertical centering. I also learned the difference between `justify-content`, `align-items`, and `align-content`.

Example:

```css
body{
  min-height:100vh;
  display:flex;
  justify-content:center;
  align-items:center;
}
```

I also learned how `100vh` helps create full-screen layouts and how `box-shadow` improves UI design.
